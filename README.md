# Todo List App

## About the Project

This is a simple Todo List application built using React. It allows users to add tasks, mark them as completed, and delete tasks when its no longer needed. I made this project to understand the basics of React.

## Features

- Add new tasks
- Mark tasks as completed
- Delete tasks
- Simple and clean user interface

## Technologies Used

- React
- JavaScript
- HTML
- CSS

## What I Learned

While building this project, I learned how to:
- Create React components
- Use the `useState` hook
- Pass data using props
- Handle button clicks and user input
- Display lists using `map()`
